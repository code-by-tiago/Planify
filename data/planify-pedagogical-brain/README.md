# Planify Pedagogical Brain — Base Universal Inicial

Esta base não é uma cópia da internet. É uma base pedagógica estrutural original para enriquecer o motor do Planify com contratos de material, padrões de questão, rubricas, progressão por etapa, lentes por disciplina e regras rígidas de qualidade.

Uso recomendado:
1. Usar como banco interno de referência.
2. Integrar ao motor antes da chamada de IA para planejar a estrutura.
3. Integrar após a geração para auditar e rejeitar conteúdo fraco.
4. Complementar futuramente com fontes abertas, domínio público, BNCC e materiais autorizados pelo admin.

Atenção: nenhum banco estático substitui completamente a IA para temas ilimitados. Ele reduz falhas estruturais e aumenta qualidade, mas conteúdo específico e original para qualquer tema ainda precisa de geração, curadoria ou fonte licenciada específica.

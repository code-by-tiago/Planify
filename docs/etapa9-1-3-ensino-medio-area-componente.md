# Planify — Etapa 9.1.3 — Ensino Médio com área + componente

## Ajuste

No Ensino Médio, a interface agora separa:

```text
Área do conhecimento
Componente/Disciplina
```

## Áreas

```text
Linguagens e suas Tecnologias
Matemática e suas Tecnologias
Ciências da Natureza e suas Tecnologias
Ciências Humanas e Sociais Aplicadas
```

## Componentes

```text
Linguagens: Língua Portuguesa, Arte, Educação Física, Língua Inglesa
Matemática: Matemática
Natureza: Biologia, Física, Química
Humanas: História, Geografia, Filosofia, Sociologia
```

## Compatibilidade BNCC

Para buscar habilidades no Ensino Médio, o sistema envia a área como filtro principal e também envia a disciplina como dado auxiliar.

## Compatibilidade Gemini

Para gerar com IA, o componente é enviado como:

```text
Disciplina • Área do conhecimento
```

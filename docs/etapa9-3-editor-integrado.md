# Planify — Etapa 9.3 — Editor integrado

## O que foi feito

```text
Planejamentos gerados com IA podem ser enviados para /editor.
Materiais gerados com IA podem ser enviados para /editor.
O editor carrega o último documento enviado.
O editor permite editar título, subtítulo e conteúdo.
O editor salva localmente por enquanto.
```

## Ainda não faz

```text
Não salva no Supabase.
Não gera DOCX.
Não envia para Google Drive/Classroom.
```

## Fluxo esperado

```text
1. Gere um planejamento ou material.
2. Clique em Enviar para Editor.
3. O /editor abre com conteúdo preenchido.
4. Edite o documento.
5. Salve localmente ou copie o conteúdo.
```

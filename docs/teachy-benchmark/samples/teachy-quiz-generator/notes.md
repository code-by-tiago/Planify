# Teachy — Lista de Exercícios (`?aiTool=quiz-generator`)

- **URL pública:** https://www.teachy.com.br/pt-BR/ferramentas/gerador-quiz
- **URL logada:** https://www.teachy.com.br/pt-BR/home?aiTool=quiz-generator
- **Créditos:** 100 por geração (botão "Criar lista")

## Formulário (capturado ao vivo)

| Campo | Observação |
|-------|------------|
| Disciplina | Combobox obrigatório |
| Ano escolar | Combobox (ex.: 5º ano) |
| Assunto | Autocomplete BNCC rico — ex.: "Frações: Representação (Matemática)" |
| Quantidade | Botões **5 / 10 / 15 / 20** + Personalizado |
| Anexos | Até 5 imagens e 3 PDFs |
| Opções avançadas | Colapsável |

## Fluxo pós-submit

1. Modal **"Como você quer criar?"** → Lista automática vs Banco de questões
2. Após escolha: quantidade + CTA **Criar lista**

## Documento gerado (padrão esperado)

- Cabeçalho compacto (disciplina, série)
- Questões em cards numerados, enunciado direto
- Gabarito separado, respostas curtas
- Sem blocos pedagógicos longos no corpo do aluno

## Benchmark theme

Matemática · 5º ano · Frações · 10 itens · com gabarito

# Criativos de lançamento Planify

## Gerar PNGs

```bash
node scripts/render-launch-social.mjs
```

Requisito: Google Chrome instalado (ou `CHROME_EXECUTABLE_PATH`).

## Dimensões (padrão oficial)

| Arquivo | Uso | Dimensão |
|---------|-----|----------|
| `carrossel-lancamento/slide-*-1080x1080.png` | Instagram/Facebook carrossel, feed quadrado | **1080 × 1080** |
| `feed-portrait/post-portrait-1080x1350.png` | Instagram feed retrato (4:5) | **1080 × 1350** |
| `reel-lancamento/reel-cover-1080x1920.png` | Reels, Stories (IG/FB) | **1080 × 1920** |
| `facebook/post-link-1200x630.png` | Facebook post com link | **1200 × 630** |
| `whatsapp-status/status-*-1080x1920.png` | WhatsApp Status (3 telas) | **1080 × 1920** |
